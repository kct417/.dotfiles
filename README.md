# GNU Stow Dotfiles

## Requirements

- Scripts assume a Unix-like environment (Linux, macOS, WSL).
- The repository must be cloned to `~/.dotfiles`.

## Usage

```
git clone --depth=1 https://github.com/kct417/.dotfiles.git
```

```
cd .dotfiles
source bootstrap.sh
```

### Neovim

```
:TransparentEnable
```

```
:Copilot setup
```
